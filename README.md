# Form exercise 

Practical exercise to understand the use of JavaScript in frontend web development.

This project will cover fundamental concepts related to DOM manipulation, API consumption, and data persistence in the browser.
---

## 🚀 Technologies Used

- HTML5
- CSS3
- Bootstrap 5
- JavaScript

---

## 📚 Features

- Carousel 
- Hero section
- Food cards with hover effects
- Bootstrap alerts
- Modern dark theme UI
- Responsive layout using Bootstrap Grid
